# nabdcode (نبض الكود)

**نبض الكود (nabdcode)** هو وكيل برمجة ذكي ومصمم واجهات مستخدم طليعي مخصص لبيئة **Termux**. تم تصميمه ليوفر للمطورين قوة الذكاء الاصطناعي مباشرة في محطة الطرفية الخاصة بهواتفهم، مع التركيز على الأداء العالي والجماليات البصرية البسيطة.

## المميزات
- **الذكاء الاصطناعي المدمج:** مدعوم بـ OpenAI GPT-4o لتقديم حلول برمجية وتصميمية متقدمة.
- **خفيف الوزن:** مصمم خصيصاً ليعمل بكفاءة على الموارد المحدودة للهواتف.
- **واجهة CLI أنيقة:** استخدام ألوان وتنسيقات جذابة لتسهيل القراءة والتفاعل.
- **دعم اللغة العربية:** واجهة وأوامر تدعم اللغة العربية بشكل كامل.

## متطلبات التشغيل في Termux
1. تثبيت Node.js:
   ```bash
   pkg install nodejs
   ```
2. تثبيت Git:
   ```bash
   pkg install git
   ```

## التثبيت
1. قم بتحميل المستودع:
   ```bash
   git clone https://github.com/your-username/nabdcode.git
   cd nabdcode
   ```
2. تثبيت التبعيات:
   ```bash
   npm install
   ```
3. بناء المشروع:
   ```bash
   npm run build
   ```
4. التثبيت عالمياً:
   ```bash
   npm install -g .
   ```

## الإعداد
يجب إعداد مفتاح API الخاص بـ OpenAI في بيئة العمل:
```bash
export OPENAI_API_KEY='your-api-key-here'
```

## الاستخدام
- لتهيئة مشروع جديد:
  ```bash
  nabdcode init
  ```
- لطرح سؤال برمي أو تصميمي:
  ```bash
  nabdcode ask "كيف يمكنني تحسين المساحات البيضاء في واجهة React؟"
  ```
- لوضع خطة لمهمة معينة:
  ```bash
  nabdcode plan "بناء مكون Navbar متجاوب باستخدام Tailwind CSS"
  ```

## فلسفة التصميم
تتبع الأداة فلسفة **"الحد الأدنى المتعمد" (Intentional Minimalism)**، حيث يتم حساب غرض كل عنصر بدقة، مع التركيز على أن التقليل هو قمة الرقي.
